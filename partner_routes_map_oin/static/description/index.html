Route Map of Partners

This module allows us to open an itinerary map from the start address of the logged-in user to the addresses of the selected partners from the list.

It will open a pop-up when we select an option from the Action after the selection of partners. We can see all the selected partners in pop-up and can also do arrangement as per our route plan.

It will use the latitude and longitude to localize the partner if this information is present on the partner.

Route Map Contacts

Select contacts to open routes on the map. Make sure the latitude and longitude information is present on the partner.

List of Selected Contacts

The system will open a pop-up to see the list of selected contacts and can also allow us to make an arrangement as per our route plan. We can also select more contacts in the list if required.

Google Route Map

This image shows, the routes from the start address of the logged-in user to the addresses of the selected partners from the list.
